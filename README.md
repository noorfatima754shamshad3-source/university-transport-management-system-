# university-transport-management-system-
A University Management System using  (OOP) manages students, teachers, courses, and academic records through classes and objects. It applies OOP concepts such as encapsulation, inheritance, polymorphism, and abstraction to organize data and automate university operations. This approach makes the system secure, efficient, reusable.
this is my 2 semester Object-Oriented Programming (OOP) project which is based  on The University Transport Management System🚎🚌 , I have created a C++ application developed to manage university transportation services efficiently. It allows administrators to manage vehicles, routes, drivers, and student registrations through a centralized system. The project improves record management, route allocation, fee tracking, and report generation.
